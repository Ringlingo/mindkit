## Deep Knowledge (Load on demand)

### Common Pitfalls
- Overfitting to training data
- Data leakage in cross-validation
- Confusing correlation with causation
- Ignoring class imbalance

### Model Selection Heuristics
- Small data (< 1K): Start with simpler models
- Large data (> 1M): Can try deep learning
- Interpretability required: Linear/logistic regression, decision trees
- Performance priority: Ensemble methods
